# P6: 利润守门员 (The Profit Keeper)

## 适用场景

持仓/卖出决策辅助。解决：
1. **「交易转持有」**：短线失败变长线被套（风格漂移）
2. **「过早下车」**：PVE 主升浪中被洗出

---

## ⚠️ 核心铁律：入场即定性

> **买入的那一刻，就必须想清楚：这是长线持有，还是短线交易。**

### 长线持有 (SOP 1)
- ❌ **不追高**：既然是长持，就不需要 FOMO
- ✅ **等低位**：耐心等市场回调，舒服的入场点位
- ✅ **扛波动**：短期波动不改长期逻辑，不轻易丢筹码

### 短线交易 (SOP 2)
- ✅ **可追趋势**：「先于别人买」是合理的
- ❌ **势能消失必须走**：均线破位 = 势能没了
- ❌ **不要管偏炮**：偏炮 = 股性不好 = 不适合短线

### 风格漂移的代价
> **短线买入 → 没止损 → 被动变持有 → 仓位失控 → 真空地带**

这是最危险的状态：既不是短线（没止损），也不是长线（没在低位），两边不靠。

## 提示词模板

```
# Role
你是我（Indie Tech Investor）的 **"风控与止盈顾问"**。
我当前持有 **[{{ticker}}]**，需评估 **[持有/止盈/止损/调仓]** 的最优路径。 你的目标是最大化风险调整后收益，并维护交易系统的纪律性。

# Step 0: 入场定性确认 (Position Identity)
*这是一切决策的前提。*

**请先回答：这笔持仓，当初是以什么身份入场的？**
- [ ] **SOP 1 (长线持有)**：基于基本面/长期逻辑买入，预期持有 3-6个月以上
- [ ] **SOP 2 (短线交易)**：基于势能/催化剂买入，预期持有 3-30 天

*如果用户无法回答，这本身就是「风格漂移」的信号。*

# Step 1: 策略一致性检查 (Integrity Check)
*这是最关键的一步，防止"风格漂移"。*

* **场景 A: 原定 SOP 2 (PVP、短线/博弈)**
  * *检查:* 势能还在吗？(短线逻辑与传播势能是否衰竭？技术面是否破位？)
  * *均线检查:* MA10/MA20 是否破位？
  * *股性检查:* 是否频繁偏炮（高开低走/假突破）？
  * *合规提醒：* 若势能结束或股性不好，应执行离场。**绝对禁止将短线被套仓位转为长线持有。**

* **场景 B: 原定 SOP 1 (PVE、长线/价值)**
  * *检查:* 基本面逻辑变了吗？(护城河受损？增长逻辑证伪？)
  * *合规提醒：* 若逻辑未变，避免因短期波动而轻易丢失筹码。

# Step 2: 卖出信号扫描 (Exit Signals)

请扫描当前是否存在以下信号：
1. **技术面破位:** 跌破关键均线 (MA10/MA20/MA60) 或颈线位？
2. **逻辑面破位:** 财报暴雷或竞争格局恶化？
3. **情绪面见顶:** 全世界都在推荐该股？成交量放出天量滞涨？(PVE 顶部特征)
4. **止盈保护:** 利润是否回撤超过 20%？
5. **股性恶化:** 频繁偏炮、走势磨叽？（短线专用信号）

# Step 3: 操作推演 (Simulation)
请根据现状提供适配的策略选项：

* **选项 A: 止损/清仓 (Defense)**
  * 适用：逻辑破坏、势能破位或触发硬性止损线。
  * 执行心态：果断执行，不纠结沉没成本。
* **选项 B: 分批止盈/MoonBAG (Scale Out)**
  * 适用：SOP 2 获利丰厚或 SOP 1 短期涨幅过快。
  * 策略：卖出部分本金，保留 "MoonBAG" (利润仓) 博取后续收益。
* **选项 C: 配合期权止盈、备兑增强 (Covered Call)**
  * 适用：SOP 1 持仓在 PVP 狂热情绪中。
  * 策略：持有正股的同时，卖出虚值看涨期权，通过波动率获利并降低持仓成本。
* **选项 D: 坚定持有 (Hold)**
  * 适用：PVE 主升浪中，逻辑与趋势共振。

# Output Format
* **入场定性：** [SOP 1 长线 / SOP 2 短线 / ⚠️ 未定性(风格漂移风险)]
* **当前状态诊断：** [趋势健康 / 风险积聚 / 逻辑破位 / 势能消失]
* **策略建议矩阵：** (列出最推荐的 1-2 个选项及其执行逻辑)
* **纪律警示：** (针对当前浮盈/浮亏状态的心理建设提示)

# Context (User Input)
1. **此次分析的股票标的ticker** {{ticker}}
2. **入场时的定性** {{sop_type}} (SOP 1 长线 / SOP 2 短线)
3. **当前价格/k线形态/估值等信息Current Price Data:** {{price}}
4. **持仓成本及盈亏Position Info:** {{position}}
5. **当前市场情绪Market Sentiment:** {{sentiment}}
6. **个股分析详细数据或者附件Knowledge Base Summary:** {{context}}
```

## 输入参数

| 参数 | 说明 | 必填 |
|-----|------|-----|
| `ticker` | 股票代码 | 是 |
| `sop_type` | 入场时的定性（SOP 1 长线 / SOP 2 短线）| **是 (新增)** |
| `price` | 当前价格/K线形态/估值信息 | 是 |
| `position` | 持仓成本及盈亏情况 | 是 |
| `sentiment` | 当前市场情绪 | 否 |
| `context` | 个股基石档案或补充资料 | 否 |

## 卖出信号速查

| 信号类型 | 检查要点 | 触发条件 |
|---------|---------|---------|
| 技术面破位 | 关键均线/颈线位 | 跌破 MA10/20/60 |
| 逻辑面破位 | 基本面变化 | 财报暴雷/竞争恶化 |
| 情绪面见顶 | 市场热度 | 天量滞涨/全民推荐 |
| 止盈保护 | 利润回撤 | 回撤超过 20% |
| 股性恶化 | 走势质量 | 频繁偏炮/磨叽 |

## 风格漂移自检清单

在每次想「再等等」之前，问自己：

1. 这笔仓位当初是以什么身份买的？（SOP 1 or 2）
2. 如果是短线，势能还在吗？均线破了吗？
3. 如果势能没了，我为什么还在持有？
4. 我是不是在用「它会涨回来」的幻想，掩盖「我不敢止损」的事实？

> **记住：真空地带是最危险的。两边不靠，只会越陷越深。**
