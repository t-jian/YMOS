## 📝 使用说明

个股分析资料存放路径：
- **短线个股调研**：`obsidian投资知识库/01 - 当前交易与研究/短期势能个股/`
- **长线个股调研**：`obsidian投资知识库/02 - 长期关注与能力圈/核心个股/`

> 💡 使用「调研一下 [股票]」暗号时，AI 会根据 SOP 分类自动建议存放位置。

**AI 读取此文件后，可以**：

1. **监控异动**：
   ```
   "检测到 [你的股票A] 突破前高，进入乐观情景，
    建议继续持有让利润奔跑。"
   ```

2. **关联推送**：
   ```
   "[你的股票B] 相关新闻：某行业订单落地，
    验证你的持仓逻辑，关注明日开盘表现。"
   ```

3. **策略建议**：
   ```
   "你的 SOP 2 仓位较低，
    若市场情绪回暖，可考虑寻找机会。"
   ```
